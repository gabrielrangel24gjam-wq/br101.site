Política de Privacidade

Esta página explica como os dados enviados pelo site podem ser usados para atendimento comercial.

Coleta de informações

Ao entrar em contato pelo site, a BR101 pode receber e armazenar dados como nome, e-mail, telefone/WhatsApp, mensagem enviada, data do envio, endereço IP e informações técnicas do navegador.

Uso dos dados

As informações são utilizadas exclusivamente para responder solicitações, prestar orientação comercial, elaborar cotações, dar continuidade ao atendimento solicitado e manter registro interno dos contatos recebidos pelo site.

Armazenamento

Os dados enviados pelo formulário podem ser armazenados em banco de dados protegido da BR101 pelo tempo necessário para atendimento comercial, obrigações legais ou solicitação de remoção pelo titular.

Compartilhamento

A BR101 não vende dados pessoais. As informações podem ser compartilhadas apenas quando necessário para atendimento, cumprimento de obrigações legais ou execução do serviço solicitado.

Contato

Para solicitar atualização, correção ou remoção de dados enviados pelo site, entre em contato pelo e-mail comercial@br101.ind.br.
